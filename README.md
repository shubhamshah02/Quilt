## Features

### Non-Destructive Capture Engine
Drive capture sessions natively without pulling hidden windows to the foreground or flashing background workspaces[cite: 1]. 
* **Dynamic Content Sources:** Capture a full physical display via `SCDisplay`, target standard active windows via `SCWindow`, lock down localized coordinate bounds (`CGRect`), or spin up real-time vertical scrolling-screenshot pipelines[cite: 1].
* **Background Isolation:** Toggle back-layer capture modes using native ScreenCaptureKit filters to keep current developer focus loops completely undisturbed[cite: 1].
* **Closed-Loop Settle Detection:** Run background structural state checks before storing real canvas frames, ensuring video timelines, hover elements, or scroll buffers are completely stable[cite: 1].

### Authoritative Workspace Picker
Avoid long configurations and manual geometry calculation. The dedicated capture source hub resolves active layout boundaries across multi-monitor setups instantly[cite: 1].

<p align="center">
  <img src="docs/images/source-picker.png" alt="Quilt capture source modal showing isolated application target selections including Xcode, Finder, and Firefox" width="550" />
</p>

* **Active Window Filtering:** Isolate and list windows by process name or bundle ID across current screen spaces while discarding system menus, desktop items, and transparent overlay frames[cite: 1].
* **Instant Configuration Resuming:** Restore previous configurations from the unified **Recent** target ledger to jump straight back into recurring workspace capture tasks[cite: 1].

### Filmstrip Review & Batch Actions
Audit stored sequences side-by-side using a responsive, non-blocking preview manager that separates frame storage away from the primary render threads[cite: 1].

<p align="center">
  <img src="docs/images/review-gallery.png" alt="Sequential filmstrip gallery displaying layout thumbs with quick compilation format action buttons" width="700" />
</p>

* **Multi-Format Target Output:** Compile image series instantly into standalone high-quality **PDF assets**, compressed multi-page **ZIP archives**, multi-image folders, or loopable **GIF animations**[cite: 1].
* **Global Boundary Overrides:** Correct layout alignments across a complete document bundle simultaneously using global batch crop adjustments[cite: 1].

### Built-In Layout Annotation Canvas
Mask sensitive user credentials, highlight structural layout anomalies, or drop technical vector cues before executing file distributions[cite: 1]. 

<p align="center">
  <img src="docs/images/canvas-editor.png" alt="Quilt canvas markup view showing rich image cropping, masking, pixel blur, shapes, and text formatting layers" width="800" />
</p>

* **Low-Latency Vector Overlays:** Inject freeform pens, line pointers, dynamic curved arrows, geometric bounding boxes, editable structural text, and layout watermarks[cite: 1].
* **Localized Privacy Controls:** Shield PII fields directly inside the image stream using customizable pixelated or gaussian blur nodes before committing layouts to disk[cite: 1].
* **Comprehensive Image Transform Control:** Rotate underlying image assets 90 degrees while dynamically projecting and transforming all associated canvas drawing vectors synchronously[cite: 1].

### File Naming Matrices & Local OCR
Define standardized engineering data naming guidelines and automate information index workflows right from the unified global preferences dashboard[cite: 1].

<p align="center">
  <img src="docs/images/export-settings.png" alt="Global settings template console with token parameters and a toggle for Apple Vision OCR data compilation" width="650" />
</p>

* **Parameterized Name Templates:** Configure programmatic file outputs with string formatting parameters including `{YYYY}`, `{MM}`, `{DD}`, `{HH}`, `{mm}`, `{ss}`, `{AppName}`, `{WindowTitle}`, and required sequence trackers `{N}`[cite: 1].
* **On-Mac Text Recognition (OCR):** Leverage local Vision framework text extraction pipelines to index and build deeply searchable PDF catalogs without connecting to cloud APIs[cite: 1].
* **Variable PDF Compression:** Scale document file footprints dynamically using multi-tier image-quality compression engines built directly into the export process[cite: 1].

### Native File Sharing Formats
Take control of document replication workflows using standalone open-standard archiving boundaries[cite: 1].
* **Unified Workspace Backups:** Save complete workspace states into portable `.quiltcapture` container archives[cite: 1]. 
* **Self-Contained Bundles:** Package raw baseline asset frames, image selection toggles, and editable JSON markup session files inside a single file boundary[cite: 1].
* **Collaborative Context Drops:** Drag-and-drop structural `.quiltcapture` containers across separate host environments to recreate multi-layered annotation timelines instantly[cite: 1].

## Core Workflows

* **Local Documentation Generation:** Capture and stitch code files, environment variables, layout mockups, and database structures right inside your IDE[cite: 1].
* **Web Slide Compilation:** Convert cloud presentation panels, design frames, or layout grids into single, continuous, offline readable manuals[cite: 1].
* **Private Knowledge Indexing:** Turn highly visual, text-heavy slides into localized knowledge catalogs that are completely scannable by indexing systems[cite: 1].
* **Archival Preservation:** Generate functional personal asset backups from layout-locked viewing containers and protected digital resources[cite: 1].

## Quick Start

1. Download the latest release disk image (`.dmg`) from [quiltformac.com](https://quiltformac.com)[cite: 1].
2. Drag **Quilt.app** into your `/Applications` folder[cite: 1].
3. Launch the app and follow the onboarding prompts to grant required **Screen Recording** and **Accessibility** system permissions[cite: 1].

**System Boundaries:** Optimized for macOS 14.0+ (Sonoma) up to macOS 26+. Built natively for both Intel and Apple Silicon architectures[cite: 1].

## Security & Architectural Guarantees

* **Zero Cloud Dependence:** Image data streams, local workspace calculations, rendering pipelines, and text recognition (OCR) actions take place entirely within your machine's hardware loop[cite: 1].
* **Strict Application Safety:** Implements secure sandbox runtime exception guidelines, checks process signing structures, and prevents path traversal inside file operations[cite: 1].
* **TCC Permission Safety:** Leverages standard macOS accessibility and Screen Capture authorization models, providing absolute runtime security visibility[cite: 1].
